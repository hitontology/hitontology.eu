A Health IT Ontology for systematically describing application systems and software products in health IT.

Introduction

There is a variety of libre/free and open-source software (LIFOSS) products for medicine and health care. The Health Information Technology Ontology (HITO) allows describing those products, as well as studies evaluating them, systematically with a defined set of classes and their relationships. With the help of linked or integrated catalogues for languages, programming languages, licenses, features, and enterprise functions, the functionalities of LIFOSS can be precisely described and compared. We publish HITO and the LIFOSS descriptions as Linked Open Data (LOD); they can be queried and viewed using Semantic Web tools, such as an Resource Description Framework (RDF) browser, SPARQL queries, and different searches, to serve the use cases given below.
